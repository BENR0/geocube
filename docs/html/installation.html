Installation¶

Stable release¶

. Use pip to install the binary wheels on PyPI:

pip install geocube
  1. Use conda with the conda-forge channel:
conda install -c conda-forge geocube

From source¶

The source for geocube can be downloaded from the GitHub repo.

$ git clone git@github.com:corteva/geocube.git

$ cd geocube
$ python setup.py install
